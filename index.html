歡迎來到我的新網誌！這是一篇測試文章

這是我親手用 Hexo 與 GitHub Pages 架設的個人部落格！這篇文章是用來測試網誌的排版與字體顯示是否正常。當你看到這行字，代表我的本地端測試已經跨出了成功的一大步。

🚀 關於這個網誌

這個網誌未來將會記錄我的生活、學習筆記或是任何感興趣的事物。託管在 GitHub 雲端上意味著它：

  • 24 小時不關機：任何時候、任何人都能連進來看。
  • 完全免費：不用花費任何一毛主機費與維護費。
  • 載入速度極快：因為它是全靜態網頁。

🛠️ 技術排版測試

1. 程式碼區塊測試 (Code Block)

    print("Hello, World! 網誌架設成功！")
